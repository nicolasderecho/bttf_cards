# Application
This Application is hosted [Here on Heroku](https://bttfcards.herokuapp.com/)

![cards](https://user-images.githubusercontent.com/4305276/94507556-609deb00-01e6-11eb-892a-90bc7cd1f6dd.png)

It's basically a simple app listing the cards from [Back to the future the card game](https://www.looneylabs.com/games/back-future-card-game) with the option to translate the card content to spanish by clicking on the Argentina flag.

It's useful if you have the game but not all the players know english as it happens with some of my friends (which is the reason that inspired this project).
